# TalentScout AI Hiring Assistant

## 📌 Project Overview
TalentScout AI Hiring Assistant is an intelligent chatbot designed to help in the initial screening of candidates. The chatbot collects candidate details and generates technical questions based on the candidate's selected tech stack.

The application is built using Streamlit and Python. It simulates an AI-based recruitment assistant that helps recruiters filter candidates efficiently.

---

## 🚀 Features

### ✅ Candidate Information Collection
The chatbot collects the following details:
- Full Name
- Email Address
- Phone Number
- Years of Experience
- Desired Job Role
- Current Location
- Tech Stack

### ✅ Technical Question Generation
Based on the candidate's selected technologies, the chatbot generates relevant screening questions.

### ✅ Answer Evaluation
The chatbot evaluates candidate responses using basic AI logic and provides feedback.

### ✅ Data Storage
All candidate data and answers are stored in a CSV file for further HR review.

### ✅ User Friendly Interface
The chatbot provides a simple and interactive UI using Streamlit.

---

## 🛠️ Technologies Used
- Python
- Streamlit
- CSV File Handling
- Basic AI Evaluation Logic
